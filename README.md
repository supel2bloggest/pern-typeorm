# pern-typeorm
PERN with typeorm docker-compose

# get-started

go to server folder\
<code>
cd server
</code>

install package\
<code>
yarn install
</code>

migration script\
<code>
yarn typeorm migration:run
</code>

seeding data\
<code>
yarn seed:run
</code>

run code\
<code>
yarn dev
</code>

# Docker add project with docker-compose 
go to main folder\
<code>
cd ../
</code>

run docker-compose 
<code>
docker-compose up
</code>

NOTE: please install nodejs and docker before run this project
